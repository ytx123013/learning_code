# 函数
函数是一个数集到数集到映射，记作
$$y = f(x)$$

## 六类基本函数
1. 常值函数 $y = C$
2. 幂函数 $y = x^a , a > 0$
3. 指数函数 $y = a^x , a > 0$
4. 对数函数 $y = \log_{a}{x}, a > 0,a \neq 1$
5. 三角函数 $y = \sin{x},y = \cos{x},y = \tan{x},y = \cot{x},y = \sec{x}$
6. 反三角函数 $y = \arcsin{x},y = \arccos{x},y = \arctan{x},y = arccot\ x$

## 反函数
如果映射$f$是双射，那么$f$可逆，记作$f^{-1}$
$$
f(a) = b \to 
f^{-1}(b) = a
$$

## 几个特殊函数
#### 符号函数
$$
y = sgn(x) = 
\begin{equation}
\begin{cases}
1\ \ \ \  x>0 \\\\
0\ \ \ \ x = 0 \\\\
-1\ \ \ \ x<0
\end{cases}
\end{equation}
$$

#### 高斯取整函数
向下取整
$$y = [x]$$

#### 狄利克雷函数
$$y = D(x) =
\begin{equation}
\begin{cases}
1\ \ \ \ x \in Q \\\\
0\ \ \ \ others
\end{cases}
\end{equation}
$$
