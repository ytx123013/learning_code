# 罗尔中值定理
假设$f(x)$在区间$[a,b]$上连续，且$f(a) = f(b)$，又假设$f(x)$在区间$(a,b)$上可导，则必然$\exists c \in (a,b)$使得
$$f'(c) = 0$$

# 拉格朗日微分中值定理
设函数$f(x)$在区间$[a,b]$上连续且可导，则必然$\exists c \in (a,b)$使得
$$f'(c) = \frac{f(b) - f(a)}{b-a}$$

# 柯西微分中值定理
设函数$f(x)$在区间$[a,b]$上连续且可导，则必然$\exists c \in (a,b)$使得
$$\frac{f'(c)}{g'(c)} = \frac{f(b)-f(a)}{g(b) - g(a)}$$
