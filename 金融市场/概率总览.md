# 概率总览
###基本知识
P表示某一事件概率，且\\(0 \leq P \leq 1\\)。
独立性，即每次实验结果都与其它实验结果没有关系。比如投掷一枚硬币正面朝上的面，第一次的实验结果不会影响第二次实验的结果。
若事件A与事件B相互独立，则同时发生事件A与事件B同时发生的概率等于两个事件发生的频率的乘积\\(P(AB) = P(A)*P(B)\\)
